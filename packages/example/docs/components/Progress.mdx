# Progress 进度条 - 待开发

#### label 用于写文字，color 指定颜色，textColor 用于指定文字颜色（但是现在不知道为什么不变）

```js
<Progress color="bg-green-500" label="40%" percent="1">
  Progress
</Progress>
```

<DemoBlock>

#### percent 可以指代进度，0 为无进度，1 为 1/4，2 为 1/2 ，3 为 3/4 ，4 为满

```js
<Progress color="bg-green-500" percent="2" textColor="green">
  Progress
</Progress>
```

</DemoBlock>
