import { CountDown } from '@dance-ui/ui'

# CountDown 倒计时

#### 1. 基本用法

```js
<CountDown time={10}></CountDown>
```

<DemoBlock>
  <CountDown time={10}></CountDown>
</DemoBlock>

#### 2. 自定义格式化方式，默认`HH:mm:ss`

```js
<CountDown size="small" time={100000} format="HH时 mm分 ss秒"></CountDown>
```

<DemoBlock>
  <CountDown size="small" time={100000} format="HH时 mm分 ss秒"></CountDown>
</DemoBlock>

#### 3. 可切换字体大小，onFinish 接收一个函数，在倒计时结束时调用

```js
<CountDown size="small" time={10} onFinish={() => alert('倒计时结束')}></CountDown>
```

<DemoBlock>
  <CountDown size="small" time={10} onFinish={() => alert('倒计时结束')}></CountDown>
</DemoBlock>
