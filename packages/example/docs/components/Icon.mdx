# Icon 图标

[IconFont 图标库地址](https://www.iconfont.cn/manage/index?manage_type=myprojects&projectId=3868223)

利用 iconfont 下载下来的 js，以 symbol 形式引用，因此会根据当前字体颜色和大小改变 icon 的颜色和大小
